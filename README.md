# Cinépolis Natal API - Sistema de Reservas 🍿

API desenvolvida como parte do processo seletivo para a vaga de Estágio em Backend. O sistema gerencia o fluxo de reservas de ingressos, desde a listagem de filmes até a gestão de sessões e assentos.

## 🚀 Status do Projeto
O projeto cumpre todos os requisitos obrigatórios para o nível de **Estágio**, incluindo:
- [x] Autenticação JWT (CASE 1)
- [x] Listagem de Filmes (CASE 2)
- [x] Listagem de Sessões por Filme (CASE 3)
- [x] Conexão com Banco de Dados Relacional (PostgreSQL)
- [x] Paginação obrigatória em todos os endpoints

## 🛠️ Tecnologias Utilizadas
- **Python 3.12**
- **Django 5.2 & Django Rest Framework**
- **Poetry** (Gerenciamento de dependências)
- **PostgreSQL** (Banco de dados principal)
- **Redis** (Preparado para Cache/Locking)
- **drf-spectacular** (Documentação Swagger/OpenAPI)
